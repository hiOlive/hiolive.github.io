MarkDown 语法样本

1. 标题语法

标题=-的数量必须大于等于2个

2. 加粗

很多语法前都可以加入加粗标记,来设置文字大小

3. 粗斜体标亮

以*或者_来标记.二者是等价的.下面以*为例说明.
一个*号(*)是斜体.两个*号(**)是粗体,三个*号(***)是粗斜体
斜体文本 斜体文本
粗体文本 粗体文本
粗斜体文本 粗斜体文本
标亮

4. 列表

无序列表以,+,-开头,有序列表以1.开头.序号与内容中间必须有空格.有序列表的数 字,当前面没有加粗符号时,没有用,它自动从1开始递增(chrome markdown preview plus).
无序列表可以用星号、加号+或者连字符-
