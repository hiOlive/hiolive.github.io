Linux 安装过程总结

发表于 2017-09-23 | 分类于 linux | 阅读次数:
  1. 启动盘制作。下载Universal USB Installer。在选项的第4步中，Set a persistent file size for storing changes.设为0。我有一次就是因没没有先0,制作的启动盘不能用。
  2. 安装的过程中，不联网，不更新。
  3. 安装方式。有清空数据，硬盘上所有数据都清空；有保留原系统，添加一个系统；整个盘安装；其它，可以指定安装位置。要选其它安装方式，可以指定哪些盘格式化，哪些盘数据保存，从而可以保留上个系统的用户数据。
  4. 在下面boot，不要去先/boot的位置，直接默认硬盘就行。我有一次就是选了，导致启动不了。
  5. 更新系统。
  6. 显卡驱动修改。系统默认使用的是开源的nvidia显卡驱动。可以安装官网驱动，但不是简单的通过界面改选安装。这样会造成无法启动或者自动重复登陆的问题。正确的安装方式是，先卸载原有驱动，再安装。
  7. 鼠标共享，用同版本的synergy.
  8. 跨电脑传文件用Nitroshare.
  9. 快速启动。用Launchy。
  10. zsh 安装。oh my zsh 安装。autojump, kill, ls
  11. 文件浏览器 pantheon。
  12. vim 编译。如果不能编译，查找是哪个模块的问题，从而通过配置减去它。我编译的时候就说perl有问题，把perl相关不配置，编译通过。
  13. vim 字体。 curiver new 不正常，暂使用其它字体解决。
  14. chrome 安装。安装谷歌同步助手，同步所有插件。
  15. hosts 修改成github上的。可以访问很多国外的网站与服务。
  16. 输入法使用对中文支持更好的 fcitx
  17. 卸载不要的程序。配置阿里源。tweek tool , dconf editor
  18. 解压程序用 peazip.
  19. git 图形界面用 smartgit.
  20. cpu, mem, net用 monitor indicator.
  21. 创建系统还原点。
  22. 安装xfce桌面系统。
  23. tmux， oh my tmux 安装。 安装tmux-yank解决复制粘贴问题
  24. 安装screenkey 显示按键。
  25. 安装tilda 终端，f1从上面拉下来，很方便。
  26. 安装cairo dock。skippy-xd
  27. swap 设置
  28. vim start folder 设置
谢谢，您的支持是我过坑的动力！
